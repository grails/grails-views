Package: grails.plugin.json.builder

[Java] Interface JsonGenerator


  • public interface JsonGenerator

    Temporary fork of groovy JsonGenerator until Groovy 2.5.0 is out. Generates JSON from objects. The Options builder can be used to configure an instance of a JsonGenerator.
